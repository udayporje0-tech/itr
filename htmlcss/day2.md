# Day 2

## Objective

To understand the basics of HTML and CSS and set up the learning resources for frontend development.

## Activities Performed

### 1. HTML Basics
- Learned the basic structure of an HTML document.
- Understood the purpose of html, head, title and body tags.

### 2. CSS Introduction
- Learned the role of CSS in web development.
- Understood how CSS is used for styling web pages.

### 3. Frontend Repository Setup
- Cloned the frontend repository shared by the mentor.
- Explored the project structure and available programs.

### 4. Box Model Introduction
- Learned the concept of Content, Padding, Border and Margin.

### 5. Flexbox Introduction
- Learned the basic purpose of Flexbox for layout design.
- Explored Flexbox learning resources.

### 6. Documentation
- Maintained daily internship documentation using Markdown files.

## Learning Outcome

- Understood the basic structure of HTML documents.
- Learned the fundamentals of CSS.
- Gained introductory knowledge of Box Model and Flexbox.
- Learned how to manage learning resources using GitHub.
